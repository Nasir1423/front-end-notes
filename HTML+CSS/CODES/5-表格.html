HTML 中的表格

表格=标题+头部+主体+脚注
  • 表格由 table 标签表示
  • 标题由 caption 标签表示
  • 头部由 thead（table head） 标签表示
    • tr 表示行
    • th 表示单元格
  • 主体由 tbody（table body） 标签表示
    • tr 表示行
    • td 表示单元格
  • 脚注由 tfoot（table foot） 标签表示
    • tr 表示行
    • td 表示单元格
注意补充
  • table 的属性 border="1" 可以设置表格边框和单元格边框为1px；但是当取值大于1时候，只能设置表格边框，单元格边框此时为 1px
